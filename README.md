## About the Project

**Application de Blog – Laravel** est une application web permettant aux utilisateurs de créer, modifier et gérer des articles, ainsi que d’interagir via les commentaires, likes et abonnements (*follow*). Elle inclut un tableau de bord administrateur pour la gestion des utilisateurs, des articles et des commentaires avec des droits d’accès précis.

## Features

- Authentification des utilisateurs  
- CRUD complet des articles  
- Système de commentaires, likes et abonnements  
- Tableau de bord administrateur  
- Gestion des droits d’accès : modification des articles réservée à leur propriétaire  
- Suppression logique (*soft delete*) des contenus  

## Technologies

- **Backend:** PHP, Laravel, Blade  
- **Frontend:** Bootstrap, HTML, CSS  
- **Database:** MySQL  
- **Outils:** Git, GitHub, VS Code
